CanvasScript3
======
Warning: This project is still in Development.

CanvasScript3 is a client-side Javascript framework to build HTML5 canvas apps similar to FLASH/ActionScript3.

### Tested browsers ###
Opera 10.51
Chrome 5.0.375.55
Firefox 3.6.3
Internet Explorer 8 + ExplorerCanvas
